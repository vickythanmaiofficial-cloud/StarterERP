# 🚀 StarterERP – Employee Management System

StarterERP is a **role-based ERP-style web application** built using **Java, Servlets, JSP, JDBC, and MySQL**.

It demonstrates **secure authentication**, **admin-controlled CRUD operations**, and a **clean MVC architecture**, making it suitable for real-world enterprise use cases.

This project was built end-to-end to understand **backend development**, **session management**, and **database-driven applications**.

---

## 📌 Features

### 🔐 Authentication & Authorization
- Secure login system  
- Role-based access control  
- Admin and Employee roles  
- Session-based authentication  
- Unauthorized access protection using filters  

### 👨‍💼 Admin Dashboard
- View all employees  
- Add new employees  
- Edit employee details  
- Delete employees  
- Validation and success/error messages  

### 👤 Employee Dashboard
- Employee-specific dashboard  
- Displays profile information  
- Restricted access to admin features  

### 🗄️ Database Integration
- MySQL database  
- JDBC-based DAO layer  
- Clean separation of business logic  

---

## 🧱 Tech Stack

- **Language:** Java  
- **Web:** Servlets, JSP

- **Backend:** JDBC

- **Server:** Apache Tomcat  
- **Architecture:** MVC  
- **Version Control:** Git & GitHub  

---

## 🧑‍💻 How to Run the Project

### Prerequisites
- Java 8+
- Apache Tomcat 9
- MySQL
- Eclipse / IntelliJ IDEA

### Run Steps
1. Import project into IDE  
2. Configure Tomcat server  
3. Start MySQL  
4. Run project  
5. Open browser and visit:

http://localhost:8080/StarterERP/login.jsp
